Drag and Drop

Integrated with Microsoft Dynamics Navision 2017, we created DLL's with Microsoft C# in order to allow users to drag and drop files into a File Management system. Users are able to drag emails onto a box developed to process outlook messages into NAV.

Coupa Purchase Order Integration

Integrated with Microsoft Dynamics NAV we used API's provided by Coupa to retrieve and extract data through XML files. We used web services to make calls against Coupa's API, then created XMLPorts to bring that data into the database. After we processed that data into purchase order documents, automating the PO process through Coupa.

Visitor Pass Integration

Fully designed and develped a visitor login portal allowing users to register themselves into the building capturing their signature (to an NDA) in the process. Included is an admin portal that allowed certain users to review and make edits. A reporting system was developed to keep track of visitors coming in and out of the building. The solution was built off of a LAMP stack infrastructure and developed using Bootstrap and PHP.

Full Purchasing/Sales Department Integration

Custom Shipping Module
