Java: Home Security System

As part of an object oriented class we were instructed to create a home security system built strictly off of java using Eclipse IDE. The home security system allowed the user to customize a home security system, setting up different rooms and security functions. These security functions include a burglar alarm system as well as a fire alarm. The application was fully customizable as well as able to mimic certain senarios including breaking and entering and a fire. Each time an event would occur the application would log it into the system allowing the user to view it at a later time.
